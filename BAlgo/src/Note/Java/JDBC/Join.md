# Join

- 두 개 이상의 테이블을 연결해서 질의하는 것

- 종류

  - 데이터 추출되는 것에 따라

    - Inner Join

      조인을 위해 사용하는 비교 조건에 맞는 데이터만 조회
      조인 조건에 맞지 않는 데이터는 조회 되지 않는다.
      join을 할 때 outer join으로 표시 하지 않으면 기본적으로 Inner join으로 조회된다.

    - Outer Join

      조인을 위해 사용되는 비교 조건에 맞지 않는 데이터도 조회된다.

      종류

      @ left outer join	: 비교 조건에 맞지 않은 왼쪽 테이블의 데이터도 조회됨

      @ right outer join : 비교 조건에 맞지 않은 오른쪽 테이블의 데이터도 조회됨

      @ full join : 비교 조건에 맞지 않은 양쪽 테이블의 모든 데이터가 조회됨
